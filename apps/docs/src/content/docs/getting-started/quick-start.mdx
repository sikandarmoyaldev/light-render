This guide will walk you through creating your first video using {PUBLIC_SITE_SHORT_TITLE}. By the end, you'll have rendered a simple video to your local machine.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v18 or higher)
- **FFmpeg** (must be available in your system PATH)

## 1. Install the Core Package

Install the `@light-render/core` package using your preferred package manager:

<CliCommandCode action="install" command="@light-render/core" />

## 2. Create Your First Project

Create a new file named `render.ts` in your project root and add the following code:

```tsx
import { LightRender } from "@light-render/core";

// 1. Initialize the LightRender instance
const renderer = LightRender();

// 2. Define your project structure
const project = {
    composition: { width: 1920, height: 1080, fps: 30 },
    segments: [
        {
            id: 1,
            durationSeconds: 5,
            layers: [
                {
                    id: "hello-text",
                    type: "text",
                    content: "Hello, World!",
                },
            ],
        },
    ],
};

// 3. Render the video
async function main() {
    console.log("Starting render...");
    await renderer.render(project, "output.mp4");
    console.log("Render complete! Check output.mp4");
}

main().catch(console.error);
```

Once the script finishes, you will find `output.mp4` in your project directory.

## 3. Run the Script

Execute your script using a TypeScript runner like tsx:

<CliCommandCode action="run" command="tsx render.ts" />

## 4. Preview Your Project

Before rendering, you can preview your project in the browser using the `@light-render/player` component. This allows you to see exactly what your video will look like:

```tsx
import { LightRenderPlayer } from "@light-render/player";

function App() {
    return <LightRenderPlayer project={project} autoPlay={true} loop={true} />;
}
```

The player provides real-time 60fps preview with full playback controls, just like the preview below:
