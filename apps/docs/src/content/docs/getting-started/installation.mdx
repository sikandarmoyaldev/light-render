It doesn't matter what framework or package manager you are using. Installing {PUBLIC_SITE_SHORT_TITLE} effects and properties is as simple as running the `shadcn add` command followed by the URL to the effect you want to install. This URL will always be formatted as <code>{PUBLIC_SITE_URL}/r/\<effect-name\>.json</code>.

The entire install process is handled by the `shadcn` CLI tool and works identically to how you would install any other Shadcn component.

## Example

<Code code={`npx shadcn@latest add ${PUBLIC_SITE_URL}/r/fade-transition.json`} lang="bash" />
