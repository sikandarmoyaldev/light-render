Just like Shadcn, this is not a video editor. Instead it is a unified rendering engine that you can use to build your own video applications. This engine is designed to be fast and customizable, so you can easily integrate it into your projects.

Each effect in this registry is built to solve specific problems that most developers face while building video applications (such as canvas orchestration, FFmpeg encoding, etc.).

If there are any effects or properties that you would like to see added to this registry, please feel free to open an issue on the <a href={PUBLIC_GITHUB_REPO_URL}>GitHub repository</a>.
